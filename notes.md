# Gravitation

## F = G(m1*m2/r^2)
 
F : Force de gravité

G : Constante gravitationnelle

G = 6,674 × 10−11 m3 kg−1 s.

m1, m2 : Masses des deux objets

r : Distance entre les deux objets

# Mécanique

## F = ma => a = F/m

a : Accélération

F : Force appliquée

m : Masse de l'objet

# Moteur temporel 

dt : Delta time

## Méthode d'Euler (pas fou mais bien pour le début) :

Vitesse = Ancienne Vitesse + Acceleration * dt

Position = Ancienne Position + Vitesse * dt
